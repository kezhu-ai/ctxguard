// Features

Built for the agent era.

⚡

Real-time budget enforcement

Wrap any agent run with --budget=80k. ctxguard watches the session JSONL via notify and fires warn | compress | kill when effective_context crosses your threshold.

🔍

The effective_context column

No other tool shows input + cache_read + cache_creation summed. That's what your 200k window actually fights — not what the bill says.

🤖

Cross-agent from day one

First context-budget tool that works with both Claude Code and OpenAI Codex CLI. One binary, one mental model, one budget.

📊

Slice history any way you want

--by model|day|hour|file across your last 30 days. Find the hour that ate your wallet; find the session that ate your context.

🦀

Single 1.1 MB binary

No npm tree, no Docker, no daemon. cargo install ctxguard and you're done. Local-first — your session JSONL never leaves your machine.

🛡️

3 modes, your call

warn for visibility, compress to keep working with a smaller context, kill when wallet is sacred. Three policies, one CLI.

// Why this exists

Prompt caching hides the cost. Your context window can't.

Anthropic charges you 10% of input cost for cache_read tokens. So your wallet doesn't scream when a 30-minute Claude session re-reads auth.ts 30 times.

But the model does have to load every one of those tokens into its 200k context window. When the window fills mid-task, your agent stops mid-tool-call with a generic error.

That's what we built ctxguard for.
